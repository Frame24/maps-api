Стилизация контейнера <iframe> через CSS

Пример: 
<iframe class="styled-iframe" src="https://yandex.ru/map-widget/v1/?ll=37.617600%2C55.755800&z=10&l=map"></iframe>

.styled-iframe {
  width: 70%;
  height: 400px;
  border-radius: 20px;
  box-shadow: 0 4px 24px rgba(0,0,0,0.15);
  border: none;
}
