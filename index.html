Возможности Яндекс.Карт без API

Этот сайт демонстрирует, что можно и чего нельзя сделать с Яндекс.Картой без API-ключа и JavaScript API.

Что можно:

  • Задать центр и масштаб карты через URL
  • Выбрать тип слоя (схема, спутник, гибрид, пробки)
  • Показать карточку организации
  • Произвольные точки и маршруты через Map Constructor
  • Динамическое изменение карты (JS без API)
  • Стилизовать контейнер <iframe> с помощью CSS
  • Пример поиска Subway в Екатеринбурге (iframe-виджет)

Чего нельзя:

  • Поиск, динамические метки, кастомные балуны и др.
Все примеры работают без API-ключа и без загрузки JS-библиотек Яндекс.Карт.
